# Planning - Refonte GL Events Paris

Planning interactif pour le projet de refonte du site **GL Events Paris**, produit par **AVANGARDISM**.

Ref. AVM-D-2026-042 · Valable jusqu'au 01 mai 2026

## Apercu

Fichier HTML unique (`planning_FINAL_gleventsparis.html`) qui presente le planning du projet sous forme de diagramme de Gantt interactif :

### Budget 1 — Refonte (33 325 € HT)
- Kick-off fin mars → Livraison avant l'ete (mars a juillet 2026)
- UX/UI, SEO/GEO, contenus, pilotage
- Filtrage par track : Jalons, Pilotage, Ateliers, UX, UI, SEO, Contenus
- Timeline visuelle avec semaines et jalons clients

### Budget 2 — Accompagnement SEO/GEO annuel (11 550 € HT)
- Octobre 2026 → Septembre 2027 · 12 mois · Cadence degressive
- 3 phases : Lancement (3j/mois), Consolidation (2j/mois), Editorial (1j/mois)
- 5 livrables : suivi positions, visibilite LLM, optimisations on-page, contenu d'autorite, reporting mensuel

- Analytics : Microsoft Clarity
